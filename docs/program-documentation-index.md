## By Program Type

### Online Programs (76)

All online programs listed in module sections above. Each operates as an IMS DC MPP (Message Processing Program) with:
- Entry via `ENTRY 'DLITCBL' USING IO-PCB DB-PCB-1`
- IMS GU call for input message
- IMS ISRT call for output message
- MFS screen definitions for 3270 terminal display

### Batch Programs (11)

All batch programs listed in the BAT section above. Each operates as an IMS BMP (Batch Message Processing) program with:
- Checkpoint/restart via COMCKPL0
- Cursor-driven processing loops
- File I/O via DD statements in JCL

### Common/Callable Modules (16)

All common modules listed in the COM section above. Each is invoked via:
- `CALL 'program-name' USING parameter-1 parameter-2 ...`
- PROCEDURE DIVISION USING linkage section parameters

### Report Programs (14)

All report programs listed in the RPT section above.
